# FLISOL 2026 — Villa Mercedes

## ¿Qué es el FLISOL?

El **Festival Latinoamericano de Instalación de Software Libre** (FLISOL) es el evento de
difusión de Software Libre más grande de América Latina. Se realiza de forma simultánea
cada año en cientos de ciudades de toda la región, siempre el último sábado de abril.

El evento es completamente gratuito y está abierto a cualquier persona, sin importar
su nivel de conocimiento técnico. La idea central es simple: compartir el software libre
con quien quiera conocerlo.

## ¿Qué es el Software Libre?

El software libre es aquel que respeta la libertad de los usuarios. Según la definición de
la Free Software Foundation, un programa es libre si garantiza estas cuatro libertades:

0. La libertad de **usar** el programa como quieras, para cualquier propósito.
1. La libertad de **estudiar** cómo funciona el programa y adaptarlo a tus necesidades.
2. La libertad de **distribuir** copias para ayudar a otras personas.
3. La libertad de **mejorar** el programa y publicar tus mejoras, para que toda la
   comunidad se beneficie.

El acceso al código fuente es una condición indispensable para las libertades 1 y 3.

## Cómo participar

No hace falta saber de tecnología para venir. El FLISOL es para todo el mundo.

- **Venir como asistente** — aprendé, preguntá, instalá
- **Traer tu computadora** — te ayudamos a probar o instalar software libre
- **Sumate** — escribinos a través de nuestras redes [flisolvm en Instagram](https://instagram.com/flisolvm)
